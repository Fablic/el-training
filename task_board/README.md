# TaskBoard

# Versions
Ruby 2.7.2  
Rails 6.0.3  
mysql 5.6  

# Settings

- install ruby, bundler
```
rbenv install 2.7.1
gem install bundler
```

- install mysql & start server(on Mac)
```
brew install mysql@5.6
mysql.server start
```

- gem install 
```
bundle install --path 'vendor/bundle'
```

- yarn install
```
brew install yarn
yarn install
```

- DB migrate
```
rails db:migrate
```

- start rails server
```
rails s
```
